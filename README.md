# XGame
一款提供各种主流技术方案的`Unity`手游框架，包含服务端、客户端等模块，采用`C++`、`C#`、`Lua`语言开发。

### 功能：
* `C++`和`Lua`交互方案 （已完成）
* 基于`RecastNavigation`的导航网格寻路方案（已完成）
* 基于`Libuv`的网络通信框架方案（已完成）
* `RPC`通信协议方案（已完成）
* `Lua`-`C++`定时器方案（已完成）
* `Unity`调用`C++` `DLL`方案（已完成）
* `SFM`影子跟随移动方案（开发中）
* 状态同步方案（待开发）
* `LuaBT`行为树`AI`方案（开发中）
* 数据库存储和读写方案（待开发）
* `Unity`下`XLua`编程方案（待开发）
* 热更`C#`代码方案（待开发）
* `AssetsBundle`打包方案（待开发）
* 资源加载和管理方案（待开发）
* 日志输出和收集方案（待开发）
